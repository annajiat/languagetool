# الصفات وأسلوب التعجب
# like جميل =>ما أجمل
# نكتبها بالشكل:
#الصفة=أسلوب التعجب
# الجزء الأول هو الصفة 
# الجزء الثاني علامة المساواة
#الجزء الثالث: صيغة التعجب الأولى والثانية
#الجزء الرابع: رسالة الخطأ المرفقة أو الشرح
#مثال:
جميل=ما أجمل|أجمِل ب
سعيد=ما أسعد|أسعِد ب
مستعجل=ما أشدّ استعجال|اشدد باستعجال
